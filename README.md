# wissen_api

## Courses CRUD table

| HTTP  | URI          | CRUD operation       | view name       |
| ------|--------------|----------------------|-----------------|
| GET   | /courses     | list all courses   | LIST            |
| POST  |              | create a courses    |                 |
| GET   | /courses/:id | retrieve a courses by id|  DETAILS   |
| PUT   |              | Update  a courses by id|                 |
|DELETE |              | Delete  a courses by id|                 |
